# 🌆 Neon Runner (Working Title)

A 2D side-scrolling game with a clean cyberpunk aesthetic, built using **FXGL (JavaFX Game Library)**.  
The game features a minimal neon city background, smooth gameplay, and collectible currency (credits).

---

## 🎮 Features

- 🪙 **Collectible Credits**
  - Gold, Silver, Bronze variants
  - Designed for high visibility (32x32 pixel art)

- 🌃 **Procedural Cyberpunk Background**
  - Grid-based (32px) for pixel-perfect alignment
  - Layered skyline (far + mid buildings)
  - Neon glow effects and minimal visual noise

- ⚡ **Performance-Friendly Design**
  - Clean rendering (no unnecessary clutter)
  - Optimized for smooth gameplay

- 🎨 **Consistent Pixel Art Style**
  - Works seamlessly with tilesets
  - Designed for readability over realism

---

## 🧱 Tech Stack

- **Language:** Java  
- **Framework:** FXGL  
- **Graphics:** Pixel Art (32x32 grid system)  
- **Rendering:** JavaFX Canvas
